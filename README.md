# Rose Day

A personalized rose greeting page with a 3D spinning rose, animated hearts, and falling petals.

## Usage

Open `index.html` directly or host it on any static server.

**Personalize with a name:**

```
https://your-domain.com/?name=John
```

Use hyphens for multi-word names:

```
https://your-domain.com/?name=My-Love
```

Hyphens are automatically replaced with spaces in the display.
